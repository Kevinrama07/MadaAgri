# Sons de Notifications

Placez les fichiers audio suivants dans ce dossier :

- `notification.mp3` - Son par défaut
- `message.mp3` - Son pour les messages
- `collaboration.mp3` - Son pour les collaborations
- `follow.mp3` - Son pour les nouveaux abonnés
- `like.mp3` - Son pour les likes
- `comment.mp3` - Son pour les commentaires

## Recommandations

- Format : MP3 ou WAV
- Durée : 0.5 - 2 secondes
- Volume : Normalisé
- Taille : < 50KB par fichier
